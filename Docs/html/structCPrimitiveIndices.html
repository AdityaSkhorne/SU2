Detailed Description

template<class IndexType>
class CPrimitiveIndices< IndexType >

Abstract wrapper for the CIndices classes of CEulerVariable, CIncEulerVariable, etc. This should only be used when the concrete type of solver variable is not known.

Note
We are using some tricks to try to make this more efficient than virtual functions.
