strongprotected

Kinds of recordings (three different ones).

Enumerator
FULL_TAPE 

Entire derivative information for a coupled adjoint solution update.

OBJECTIVE_FUNCTION_TAPE 

Record only the dependence of the objective function w.r.t. solver variables (from all zones).

◆ Tape_Positions

enum CDiscAdjMultizoneDriver::Tape_Positions
protected

Position markers within a tape.

Enumerator
START 

Beginning of the tape.

REGISTERED 

Solver variables are registered on the tape.

DEPENDENCIES 

Derived values (e.g. gradients) are set.

OBJECTIVE_FUNCTION 

Objective function is set.

TRANSFER 

Solution data is transferred between coupled solvers of different physical zones (giving cross contributions).

ITERATION_READY 

Until here, all derivative information is gathered so that it can be connected to a solver update evaluation.
