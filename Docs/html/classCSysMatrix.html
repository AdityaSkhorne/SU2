Sets 2 blocks ij and ji (add to i* sub from j*) associated with one edge of an FVM-type sparse pattern.

Note
The parameter Overwrite allows completely writing over the current values held by the matrix (true), or updating them (false).
Parameters
[in]edge- Index of edge that connects iPoint and jPoint.
[in]block_i- Subs from ji.
